Local Mastering

Drop a lossless audio file (WAV, AIFF, or FLAC). The app will create:

  • Club Master — 48 kHz / 24-bit, target −7.5…−6.5 LUFS-I, TP ≤ −0.8 dBTP
  • Streaming Master — 44.1 kHz / 24-bit, target −10…−9 LUFS-I, TP ≤ −1.0 dBTP
  • Unlimited Premaster — 48 kHz / 24-bit, peaks ≈ −6.0 dBFS (no limiter)
